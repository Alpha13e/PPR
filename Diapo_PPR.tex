\begin{frame}
    \frametitle{Introduction}
    Les distances stellaires étant sans commune mesure avec celles terrestres, nous devons inventer de nouvelles méthodes de mesure:
\\[0.5cm]
\begin{description} % j'aime pas le rendu. manque bulet points ?
    \item[La Parallaxe] calcule la distance d'un objet par trigonométrie en mesurant son angle d'observation.
    \item[Les chandelles standard] sont des objets dont une propriété particulière permet de déterminer sa luminosité réelle,
        et donc sa distance.
    \item[L'effet Doppler] consiste à mesurer le décalage spectral d'un objet, qui est lié à la vitesse relative de l'objet par la loi de même nom. 
        Cette méthode ne sera pas traitée ici. % c'est tout ?
\end{description}
\end{frame}

\subsection{Application}
\begin{frame}
    \frametitle{La Parallaxe}
    \framesubtitle{Application}
    % Logo Gaia en marge/coté, formule erreur, mention directe sources ? comment remplir ?
    % ajouter bit apres code ! mention f (equation)
    \begin{columns}[T] % aligned with top
        \begin{column}{0.3\textwidth}
            \textbf{Source:}
            \\[8pt]
            \includegraphics{gaia_logo.png}

            Le satellite est positionné sur le point L2, mais les calculs sont ramenés pour $r=1$~ua.

            La définition du parsec étant fonction de l'arcseconde, $[d]=pc$.
        \end{column}
        \begin{column}{0.7\textwidth}
            \textbf{Erreur:}

            On a un calcul de type $d=\frac{cte}\varpi$ avec $\sigma_d$ l'incertitude sur la distance et celle sur la mesure $\sigma_\varpi$ fournie:
            \[
            d\, w=cte \Leftrightarrow \partial (d\,w)=0 \Leftrightarrow d\,\partial w=-w\,\partial d
            \]
            \begin{equation*}\label{eq.parallax-error}
                \frac{\sigma_\varpi}{\varpi}=\left|\frac{\sigma_d}{d}\right| \Longrightarrow
                \sigma_d=d\left|\frac{\sigma_\varpi}{\varpi}\right|
            \end{equation*}
            Cependant, selon \citet[Section~3.2]{DR2-parallax}, deux conditions supplémentaires s'imposent:
            \begin{equation*}
                f=\frac{\sigma_\varpi}{\varpi}<20\%
            \end{equation*}
            \begin{equation*}
                \varpi>0
            \end{equation*}
        \end{column}
    \end{columns}
\end{frame}

\begin{frame}[fragile] % étonnament permet de compiler
    \frametitle{La Parallaxe}
    \framesubtitle{Programme}
\begin{lstlisting}
# Importation des fichiers (coupee)
output.write("source_id\tparallax (mas)\tparallax_error(mas)\tdistance(pc)\tdistance_error (pc)\n")
# Automatisation
i=0 # compte les echecs
n=100 # nombre de lignes a lire, arbitraire

for _ in range(n):
    l=input.readline()
    id,par,perr,*_=l.split(',')

    par=float(par)+0.029 # erreur zero-point (-biais, biais=-0.029)
    f=abs(float(perr)/par) # calcul de la qualite
    dist=1/(par*1e-3) 

    if f<0.2 and dist>0:
        derr=dist*f # propagation de l'erreur
        output.write(f"{id}\t{par}\t{perr}\t{dist}\t{derr}\n")
    else:
        i+=1

output.close()
input.close()
print(f"Done. {i} errors over {n}: Success rate {1-i/n:.2f}.")
\end{lstlisting}
\end{frame}

\subsection{Théorie}
\begin{frame}
    \frametitle{Chandelles standard: les Céphéides}
    \framesubtitle{Théorie}
    % les 2 magnitudes, 
    \textbf{La magnitude,} unité de luminosité sans dimension:
    \begin{block}{La magnitude absolue}
        \begin{equation*}
            M=-2.5\,\log_{10}(\frac{l}{4\pi(10\,pc)^2L_{Soleil}})+k
        \end{equation*}
    \end{block}

    \begin{block}{La magnitude apparente}
        \begin{equation*}\label{eq.module_distance}
            \mu = m-M = 5\log_{10}(d)-5 
            \Longleftrightarrow d = 10^{\frac{m-M+5}{5}}
        \end{equation*}
        Magnitude mesurée à une distance $d$, liée à $M$ par le \textbf{module de distance}.
    \end{block}
    Cependant, des poussières absorbent les rayonnements, c'est \textbf{l'extinction interstellaire}.
    Pour pallier on utilise l'indice de Wesenheit. % on l'intègre pas de suite ?
    \\[8pt]
    \textit{On note que plus un astre est lumineux, plus $M$ est négatif.} % réécrire
\end{frame}

\subsection{Calibration}
\begin{frame}
    \frametitle{Chandelles standard: les Céphéides}
    \framesubtitle{Calibration}
    % liste raisons calibration qqn d'autre
    % equation ABL, mention Rippepi
    J'ai dû déléguer la calibration, utiliser celle de chercheurs car:
    \begin{itemize}
        \item Peu de céphéides sont suffisamment proches pour utiliser la parallaxe et fiables.
        \item Elles sont brillantes et de luminosité variable: on doit trouver un proxy.
        \item Les problèmes de la parallaxe s'appliquent toujours: on doit utiliser l'Astrometric Based Luminosity (ABL):
         \begin{equation*}\label{eq.ABL} % delete, concerne plus ? si.
            \text{ABL} = 10^{0.2 M} = \varpi 10^{0.2 m - 2}
         \end{equation*}
    \end{itemize}
    On a utilisé la théorie de \citet{cepheid-calibration}, mais la calibration est liée à nos données \citet{Ripepi_datapaper},
    obtenues via la plateforme VizieR \textit{(Université de Strasbourg)}.
